# CodeAlpha Image Gallery

A modern and responsive Image Gallery developed using HTML, CSS, and JavaScript as part of the CodeAlpha Frontend Development Internship.

## Features
- Responsive image gallery layout
- Next and Previous navigation buttons
- Lightbox image preview
- Hover effects and smooth transitions
- Mobile-friendly design
- Clean and attractive user interface

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Project Objective
To create an interactive image gallery that provides a smooth user experience across different devices and screen sizes.

## Future Enhancements
- Image categories and filters
- Search functionality
- Slideshow mode
- Dynamic image loading
